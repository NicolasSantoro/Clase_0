        Las BANDERAS serían puntos de control (sean STR, INT, FLOAT, no importa) y sirve para determinados casos.
        Por ejemplo, calcular máximos y mímimos en una lista de números otorgados (por el usuario, en este caso).
            Ej: bandera = False

        while True:
            numero = prompt('', 'Ingrese un Número')

            if numero == None:
                break

            numero = int(numero)

            if bandera == False:
                maximo = numero
                minimo = numero
                bandera = True
            else:
                if numero > maximo:
                    maximo = numero

                if numero < minimo:
                    minimo = numero
            
            numero = int(numero)
        
        self.txt_maximo.delete(0,tkinter.END)
        self.txt_maximo.insert(0, maximo)

        self.txt_minimo.delete(0,tkinter.END)
        self.txt_minimo.insert(0, minimo)
